Detailed Description

#include <jive/util/Array.h>

This header file exports a set of convenient typedefs, such as Vector and Matrix, that represent different types of one-dimensional and multi-dimensional arrays.

Note that this file includes the header file <jem/base/array/Array.h>.

Typedef Documentation

typedef ::jem::Array<double> jive::util::Vector

The Vector type represents a one-dimensional array of double precision numbers. It is an alias for jem::Array<double>.

Examples:
Database.cpp, and meshgen.cpp.
